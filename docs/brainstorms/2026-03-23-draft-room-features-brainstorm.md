# Draft Room Missing Features

## What We're Building

Five improvements to the snake draft room: a bug fix for odd-round picks, a team viewer panel, player rankings with auto-pick, draft completion state, and a viewer mode for spectators.

## Decisions

### 1. Bug: Odd Round Pick Not Working

**Symptom:** UI shows "Your Turn" on odd rounds, but clicking Pick does nothing. Admin can still force-pick.

**Fix:** Round indexing mismatch between display logic and server-side pick validation in `calculate_snake_turn()`. Likely 0-indexed vs 1-indexed inconsistency.

---

### 2. Team Viewer (replacing "My Team" sidebar)

**What:** Replace the "My Team" section in the right sidebar with a dropdown-controlled team viewer.

- Default selection: current user's team ("My Team")
- Dropdown lists all users in the season
- Selecting a user swaps the player list below to show their picks
- Same list style as current My Team (player name, role, IPL team, round)

**Why:** Simpler than tiles + modal. Reuses existing UI. Scales to 10 users with just a dropdown.

---

### 3. Player Rankings + Auto-Pick

**Rankings data:**
- Add `ranking` column to players table
- Admin imports ranking via CSV upload

**Display changes:**
- Default sort in Available Players list = by ranking (ascending)
- Show rank number on each player row

**Auto-pick on timer expiry:**
- Pick the highest-ranked available player (pure ranking, no role balancing)

---

### 4. Draft Completion State

**What:** When all rounds are done:
1. Show "Draft Complete!" banner in the draft room
2. Show a button to navigate to the league page

No full team summary needed — just the message and CTA.

---

### 5. Viewer Mode

**What:** Authenticated users can view any draft via a shareable link, without being a season participant.

- Requires login (authenticated users only)
- Read-only — no pick functionality
- Simplified UI: draft board (picks by round/team) + current turn indicator + timer
- No sidebar (no Available Players, no My Team/Team Viewer)
